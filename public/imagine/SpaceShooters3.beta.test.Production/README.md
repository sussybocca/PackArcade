# SpaceShooters 3

## Production Build - Immersive Open World Space Shooter

### Setup Instructions
1. Place your audio files in `assets/audio/`:
   - `background-music.mp3`
   - `laser.mp3`
   - `explosion.mp3`
   - `game-over.mp3`

2. Place your video files in `assets/video/`:
   - `player.mp4` (animated player spaceship)
   - `enemy1.mp4` (animated alien type 1)
   - `enemy2.mp4` (animated alien type 2)
